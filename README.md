
Мессенджер на HTML/CSS/jQuery/PHP/MySQL


Есть функции:
- регистрации
- поиска диалогов
- проверки логина и пароля
- создание диалогов
- удаление диалогов
- удаление сообщений
- отправки сообщений
- автоматического создания всех БД при их отсутствии
- без перехода по страницам, все в одном окне
- добавлена функция чата - favourites ( Избранное )
- добавлена защита от CSRF-атак
- все реализовано на ООП

Пока в базовом варианте, реализовано на Long Poliing, в дальнейшем планируется переход на сокеты
Проект исключительно для учебных целей. Шифрование не используется

Настройка баз данных: connections.php
