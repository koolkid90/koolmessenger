
Мессенджер на HTML/CSS/jQuery/PHP/MySQL


![algo](demo.jpg)

Есть функции:
- регистрации
- поиска диалогов
- проверки логина и пароля
- создание диалогов
- удаление диалогов
- удаление сообщений
- отправки сообщений
- автоматического создания всех БД при их отсутствии
- без перехода по страницам, все в одном окне ( jQuery )
- добавлена функция чата-favourites
- добавлена защита от CSRF-атак
- реализованы ООП, композиция, архитектура

Проект для учебных целей. Шифрование не используется

Пока в базовом варианте, реализовано на Long Poliing, в дальнейшем планируется переход на сокеты

Настройки БД: settings.php
